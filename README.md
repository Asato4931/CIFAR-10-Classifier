# CIFAR-10-Classifier


## 概要

CIFAR-10 画像データセットを用いて、画像分類モデルを作成しました。

① 4-Layer-CNN : Pytorchを用いてアーキテクチャを0から構築

② Efficient Net Finetune : Efficient Net V2をFinetune

の2種類あります。
