# CIFAR-10-Classifier


## 概要

CIFAR-10 画像データセットを用いて、画像分類モデルを作成しました。

- 4_Layer_Classifier.ipynb : Pytorchを用いてアーキテクチャを0から構築

- efficient-net-fine-tune.ipynb : Efficient Net V2をFinetune

の2種類ございます。
